ESD Mini Project

This locking system uses WiFi and any WiFi capable device to unlock the lock, the main module of the system is an ESP8266-12e which is housed in a nodemcu devolopment enviroment

This was supposed to be a Door lock which could be unlocked using an Rfid card but due to the sad and un-timely demise of two rfid cards
