Setup Stage

Sets the User defined password to the ESP8266-12e

  • This can be only done once during the setup process.
  • The device password is coded into the code itself, so the password is retained after a reset.
  • This is not the case for the User define password which can be updated and also reseted
  • The device password cannot be used to unlock the device
